## 🚀 About

Hide in Sky is a **premium, sleek desktop utility** that allows you to securely conceal sensitive data, documents, and archives inside standard, viewable image files.

By leveraging advanced file-fusion techniques, the output behaves exactly like a normal **JPEG image**, while secretly containing a fully intact **ZIP payload**.

## 🚀 How to Use

### 1. Open the Application

- Double-click `HideInSky.exe`

### 2. Windows Security Warning (Important)

When you open the app, you may see:

**"Windows protected your PC"**

This happens because the application is **not digitally signed**.

### 💡 Why this happens:

- Code-signing certificates are **very expensive**
- Independent / non-commercial developers often distribute apps without them
- Your system shows this warning as a precaution

👉 **Your data is safe** — the app runs 100% locally and does not connect to the internet

---

### ✅ How to Proceed

1. Click **More info**
2. Click **Run anyway**

---

### 3. Using the Tool

Once the app opens:

#### 🔐 Hide Files

1. Select an **Image (.jpg)**
2. Select a **ZIP file**
3. Click **Merge**
4. Output: Image with hidden data
